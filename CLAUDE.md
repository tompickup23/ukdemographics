# UK Demographics — Claude Code Context

## Overview
Population data for every community. Ethnic projections, school data, housing demand, and demographic change across 320 local authorities.

**Stack**: JavaScript | HTML | CSS
**Hosting**: GitHub Pages
**Data**: ONS Census, school census, housing data
**Branch**: main

## Key Patterns
- Static site with JavaScript data visualisation
- JSON data files for demographic statistics
- Charts showing population trends, ethnic breakdown, projections
- Local authority level granularity (320 areas)

## Data Sources
- ONS Census 2021
- School census data
- Housing demand statistics
- Population projections by ethnicity

## Rules
- All data must cite ONS or official source
- Numbers must be accurate — cross-reference with source
- Mobile-responsive charts and tables
- Accessible colour schemes for charts
- British English throughout
- Keep data files under version control
